📋 Lógica del Sistema Inteligente de Refuerzos

  • Grosor inicial: Siempre comienza en 8mm (código 2)
  • Ratio objetivo: Mínimo 4.0 de seguridad
  • Sistema de refuerzos automático:
    1. Probar SIN refuerzos → Si ratio ≥ 4.0 → Válido ✓
    2. Si ratio < 4.0 → Añadir PERIMETRAL (+1.0 al ratio)
    3. Si aún ratio < 4.0 → Añadir TIRANTE (+1.5 adicional al ratio)
    4. Si con ambos NO llega a 4.0 → Pasar al siguiente grosor y repetir
  • Rango de medidas: Desde 80×50×50 hasta 250×100×90 (igual que abiertos)
  • Columnas CSV: Incluye Perimetral y Tirante (Sí/No)
  • Nota: Luego eliminarás manualmente los que no necesiten refuerzos
